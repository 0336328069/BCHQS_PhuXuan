# Thư mục hình ảnh

Đặt 3 ảnh banner vào đây với tên:
- `banner-1.jpg` hoặc `banner-1.png`
- `banner-2.jpg` hoặc `banner-2.png`
- `banner-3.jpg` hoặc `banner-3.png`

Sau đó cập nhật đường dẫn trong `app.vue`:
```typescript
const bannerSlides = [
  { image: '/images/banner-1.jpg', title: 'Hoạt động huấn luyện' },
  { image: '/images/banner-2.jpg', title: 'Phòng chống thiên tai' },
  { image: '/images/banner-3.jpg', title: 'Sẵn sàng chiến đấu' }
]
```
